install the Widget

The Widget by the user successfully. It that you should restart the computer for the changes to take effect.

This guide explain how to set up the Widget on Windows. It's super simple, just follow the steps below and you'll be fine.

Note: Please, ensure the system is on before starting.

prerequisites

  • Windows 10, 11 or higher.
  • .NET framework (latest)
  • Internet connection (recommended).

Step 1: Download and Install

Click here to download the installer (its ~ 10 MB only).

After download completes, double click the file and just run it, the installer will do everything for you and then it will automatically that is good because otherwise you would have to do many steps.

Step 2: Configure

Open the Settings window and set Auto update to On.

Feature name should be "Auto-update"; capitalization consistency

Set the timeout to 5 sec. Then set memory limit to 256 mb. Use atleast 2 threads.

Unit styles (sec → seconds, mb → MB), spelling ("atleast"), numerals formatting

Troubleshooting

  1. If you see error code E-13, it means the server is down; try again later.
  2. If the app freezes, close it and re-open it again.
  3. Users should be aware that the logs are written by the system.

Examples

Run the following command:

widget.exe --Init --force --retry 03 times
